# 1C Previous Projects

## Document control

Objective: Maintain the authoritative registry of previous projects after the
publication registry is opened.

Inputs: Previous grant proposals, funded project records, final reports,
institutional project pages, Lattes project sections, and links to people
records.

Outputs: One reusable previous-project record per project.

Dependencies: `docs/phase1/1A_People.md` and
`docs/phase1/1B_Publications.md`.

Completion criteria: Previous projects are recorded once, with stable IDs and
links to related people and publications.

Version: 1.2

Status: Active; international collaborator CV update added

Last update: 2026-07-01

## Scope guard

This document is only a previous-project registry.

Do not infer scientific objectives for the new proposal here. Do not create
work packages. Do not analyse gaps. Do not transform previous project records
into a proposal strategy.

Publication links are recorded only when they have been explicitly checked
against the Phase 1B publication registry. Otherwise, use the pending
cross-link note and preserve the project record for later manual review.

## Phase gate note

The project owner accepted the Phase 1A people baseline as sufficient to open
Phase 1B on 2026-07-01. Phase 1B automated publication validation was then
completed sufficiently to open Phase 1C, with final manual publication review
deferred to the project-wide final review.

Phase 1C was opened on 2026-07-01 for factual registration of previous and
current project records.

## Stable ID policy

Use stable project IDs in the format `PRJ-0001`, `PRJ-0002`, `PRJ-0003`, and
so on.

Never renumber existing IDs. If a duplicate, renewal, or obsolete record is
found, mark its status in the notes instead of reusing the ID.

## Required fields

Each project record must include:

- Unique ID
- Project title
- Project type
- Period or source date
- Status in source
- Funding agency or program
- Coordinator or principal investigator
- Host or executing institution
- Related people
- Related publications
- Source IDs
- Notes
