# 1H Bibliography

## Document control

Objective: Maintain the authoritative bibliography map for the research
program without duplicating publication metadata already recorded in Phase
1B.

Inputs: Phase 1B publication records, Phase 1F preliminary-result records,
Phase 1G idea-seed records, CV-derived publication metadata, source
inventory entries, and future proposal-support references.

Outputs: One reusable bibliographic-use record per Phase 1B publication
record, with links to the records that use it.

Dependencies: `docs/phase1/1B_Publications.md`,
`docs/phase1/1F_PreliminaryResults.md`, and
`docs/phase1/1G_Ideas.md`.

Completion criteria: Bibliographic entries are recorded once, with stable
IDs and links to the documents that use them.

Version: 1.1

Status: Active; initial bibliography map opened

Last update: 2026-07-01

## Scope guard

This document is only a bibliography map.

Do not duplicate full bibliographic metadata here. Publication title,
authors, DOI, venue, publication status, related people, source IDs, and
validation notes remain authoritative in `docs/phase1/1B_Publications.md`.

Do not analyse gaps, rank references, define scientific objectives, create
work packages, or write CNPq proposal text in this document.

Bibliography themes are classification aids only. They do not imply
priority, centrality, novelty, or final proposal use.

## Phase gate note

Phase 1H was opened on 2026-07-01 after Phase 1G created the initial
unselected, non-objective idea-seed registry.

Phase 1A through Phase 1G remain available for deferred manual refinement,
but they no longer block this initial bibliography pass.

## Stable ID policy

Use stable bibliography IDs in the format `BIB-0001`, `BIB-0002`,
`BIB-0003`, and so on.

Initial `BIB-*` IDs are assigned one-to-one to the current `PUB-*` records
in Phase 1B. Never renumber existing `BIB-*` IDs. If a publication record
is later retired, merged, or corrected, preserve the bibliography ID and
record the change in notes.

- Related Phase 1B publication ID
- Title aid
- Bibliography theme tags
- Related preliminary results
- Related idea seeds
