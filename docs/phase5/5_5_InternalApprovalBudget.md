# 5.5 Internal Approval and Budget Collection

## Document control

Objective: Record the approved checkpoint for summarized proposal circulation,
project-member approval, and budget collection before final project writing.

Inputs: Completed Project Matrix, Gap Analysis, Scientific Atlas, Team Book,
Phase 1 knowledge-base records, and project-owner instructions.

Outputs: Summarized project proposal for team circulation, approval record,
budget item collection, budget justification collection, and consolidated
budget/schedule inputs for the Master Project.

Dependencies: Completion of Phase 5 - Project Matrix.

Completion criteria: Project members have received the summarized proposal;
approval, comments, and required adjustments are recorded; budget items,
values, and justifications are collected; and the project owner accepts the
checkpoint as complete.

Version: 1.0

Status: Locked until Phase 5 is complete

Last update: 2026-07-02

## Scope guard

This checkpoint is a governance and budget-collection gate. It does not replace
Phase 6 - Master Project, Phase 7 - CNPq Proposal, or Phase 8 - Final Review.

Do not populate this document before Phase 5 is complete.

Do not use this checkpoint to bypass Phase 4 Gap Analysis, Phase 5 Project
Matrix, or the controlled writing phases.

## Checkpoint requirements

When this checkpoint opens, it must track:

- summarized proposal sent to all project members;
- member approval, comments, objections, or requested adjustments;
- budget items requested by members or institutions;
- values, categories, and justifications for each budget item;
- schedule or team information required for the consolidated project package;
- project-owner acceptance before Phase 6 starts.

## Final-output implications

Later phases must produce two coordinated final outputs:

- CNPq form blocks that stay comfortably below the character limits, using
  about two thirds of each block limit as an operational target;
- a consolidated PDF project document containing the project elements, team,
  budget, schedule, and other complete project information.
