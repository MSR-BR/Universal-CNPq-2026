## Matrix design principle

The project-owner clarification recorded in Phase 4 defines metal complexes as
the aggregating element of the project. Phase 5 should therefore organize the
matrix around metal-complex systems rather than around disconnected thematic
blocks.

Candidate matrix entries should preserve three links:

1. the metal-complex material basis;
2. the experimental or computational variable being studied;
3. the evidence, capability, and validation route that supports later project
   design.
