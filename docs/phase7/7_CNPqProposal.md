# 7 CNPq Proposal

## Document control

Objective: Prepare the CNPq proposal blocks only after the Master Project is
complete.

Inputs: Master Project, completed internal approval and budget-collection
checkpoint, and all prior controlled documents.

Outputs: CNPq-specific form blocks derived from the reusable master project.

Dependencies: Completion of Phase 6.

Completion criteria: The CNPq form blocks are drafted from the Master Project,
checked against call-specific requirements, and kept comfortably below the
maximum character limits, using about two thirds of each block limit as an
operational target.

Version: 1.1

Status: Locked until Phase 6 is complete

Last update: 2026-07-02

## Scope guard

Do not start writing the CNPq proposal before Phase 6 is complete.

Do not treat the form blocks as the only final output. The consolidated PDF
project document is also required and is checked in Phase 8.
