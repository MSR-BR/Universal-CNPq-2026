## Scope guard

This subsection is a quality-control aid. It does not select scientific
directions, define gaps, assign responsibilities, create work packages, create
facility-use commitments, or write proposal text.

Evidence bundles are navigation groupings for already recorded source-backed
evidence. They do not measure scientific importance, priority, novelty, merit,
centrality, or final proposal use.

The `EB-*` codes below are local Phase 2F shorthand. They do not replace stable
Phase 1 IDs.
