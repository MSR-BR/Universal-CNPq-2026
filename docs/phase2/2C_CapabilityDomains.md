## Scope guard

This subsection records source-backed capability signals only. It does not
select project priorities, define scientific objectives, analyse gaps, assign
responsibilities, create work packages, or write proposal text.

Capability domains are navigation labels for the evidence base. They can be
split, merged, or renamed during later manual review, but this document does
not decide which domains become part of any future document structure.
