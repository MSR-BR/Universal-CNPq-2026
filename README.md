# Universal CNPq 2026

Permanent knowledge base for the research program associated with
MSR-BR/Universal-CNPq-2026.

This repository is not only a proposal workspace for the CNPq Universal 2026
call. It is the single source of truth from which future proposals, reports,
scientific syntheses, and institutional materials can be generated.

## Document control

Objective: Define the repository purpose and point contributors to the
controlled project documents.

Inputs: Project architecture instructions and repository governance rules.

Outputs: Entry point for the permanent research-program knowledge base.

Dependencies: `docs/00_Project_Status.md`, `docs/01_Methodology.md`, and
`docs/02_Project_Roadmap.md`.

Completion criteria: The README states the repository purpose without
duplicating detailed methodology or current-status information.
